Normalized Database Schema

The dataset follows a fully relational schema consisting of 11 clean, interconnected tables.

teams

Country info, FIFA and Elo rankings, group stage configurations, and squad valuations.

venues

Stadium names, locations, capacities, timezones, coordinates, and pitch elevations.

matches

Primary schedule, scores, expected goals (xG), referee assignments, and player of the match.

squads_and_players

Full 26-man squads for all 48 countries including positions, birth dates, club listings, and market values.

match_events

Minute-by-minute events including goals, assists, yellow/red cards, penalty details, and substitutions.

match_lineups

Player-level starting lineups, positions, substitution minutes, and team captains.

match_team_stats

Detailed team performance metrics (possession, shots, passes, corners, saves, and fouls).

player_stats

Aggregated individual statistics compiled automatically after every match.

Quick Start Guide

Query the SQLite database or inspect data files in minutes.

SQLite / SQL Query 
-- Joint query: Fetch match details with venue and score
SELECT 
  m.match_id,
  ht.name AS home_team,
  at.name AS away_team,
  m.home_score || '-' || m.away_score AS score,
  v.name AS stadium,
  v.city AS city
FROM matches m
JOIN teams ht ON m.home_team_id = ht.team_id
JOIN teams at ON m.away_team_id = at.team_id
JOIN venues v ON m.venue_id = v.venue_id
ORDER BY m.match_id LIMIT 10;
Python Quick Start 
import pandas as pd
import sqlite3

# Connect to the SQLite relational database
conn = sqlite3.connect("sqlite_fifa_world_cup_2026.db")

# Load matches data into a Pandas DataFrame
matches_df = pd.read_sql_query("SELECT * FROM matches", conn)
print(matches_df.head())
